#Análise de faturamento - Estrutura do projeto
##Autor: Gabriel da cruz

###Sobre
Esse projeto é sobre análise de dados exploratória de um dataset criado pelo próprio autor acerca de vendas dos produtos de uma loja, em que os dados são atualizados dinamicamente nos dois arquivos existentes a 
cada vez que é atualizado o código. Afinal, o dataset é criado no início do projeto com a biblioteca random. Além disso, é analisado as nuances dos dados e por fim é criado um gráfico que mostra o faturamento ao
longo dos meses de 2023. 

###Feature

-> Arquivo 1) Sell_analysis

-Uso da blioteca pandas para criar o dataframe e fazer a correção dos dados, que envolve a transformação do tipo dos dados da coluna date e mes. Também, é feito a classificação, troca de dados para reajuste.

-Biblioteca datetime para transformar o tipo de dado da coluna mes para datetime.


-> Arquivo 2) Analise_faturamento
